# Travel Dashboard (by. streamlit)🛫🛫

## 1. 기본정보
* streamlit : **데이터 애플리케이션**을 쉽고 빠르게 개발할 수 있는 **파이썬 라이브러리**


* 사용방법
    1. 가상환경 설정
    2. 패키지 설치 
        - `pip install streamlit`
        - `pip install scikit-learn`
        - `pip install matplotlib`
        - `pip install plotly`
    3. 앱파일 생성  
    4. 실행 `streamlit run app_name.py`
